# Gesture Animations

## whileHover, whileTap and whileDrag
With these gesture animations, you can quickly make elements interactive. With just one line of code, you add an automatic animation between two states.

<WhileTapHoverDrag />

```js
whileTap={{
    scale: 0.8,
    backgroundColor: 'var(--color-surface-300)',
}}

whileHover={{
    border: 'solid 6px var(--color-surface-300)',
}}

whileDrag={{
    scale: 0.2,
    backgroundColor: 'var(--color-surface-300)',
    borderRadius: '100%',
}}
```

## While Focus
Only HTML input elements can have a focus state, so you have to use a `<motion.input>`.
<WhileFocus01 />

```js
whileFocus={{
    background: [
        'linear-gradient(to right, #f0f -200%, #0ff -100%, #f0f 0%,   #0ff 100%)',
        'linear-gradient(to right, #f0f -100%, #0ff 0%,    #f0f 100%, #0ff 200%)',
        'linear-gradient(to right, #f0f 0%,    #0ff 100%,  #f0f 200%, #0ff 300%)',
    ],
    transition: {
        duration: 2,
        repeat: Infinity,
        ease: 'linear',
    },
}}
```
Example from Matt Perry. If you type empty space and leave `input`, you will get a color indicator by using `onBlur` eventListener on `input`.
<WhileFocus02 />

```js 
onBlur={(e) => {
    const { value } = e.currentTarget
    if (!value) {
        setValidity('neutral')
    } else if (value.includes(' ')) {
        setValidity('invalid')
    } else {
        setValidity('valid')
    }
}}
```
